Hexo与github做个人博客

这里简单说明如何利用Hexo与Github的Pages来做个人博客。

步骤一

在github上建立一个仓库，名字为“你的github帐号名.github.io”,名字必须是这样的

步骤二 安装nodejs

​ 下载按照nodejs

步骤三 安装hexo以及管理博客

  1. 打开命令行工具 输入 npm install hexo-cli -g 安装hexo
  2. 创建一个文件夹用来作为博客空间
  3. 进入此目录后,输入hexo init
  4. 接着输入npm install 安装一些依赖包
  5. 直接输入hexo server -p 40000 这样就启动一个本地的服务器，可以访问目录生存的网页了。

步骤四 编写博客

  1. ​ 在博客目录中的source>_posts目录下建立md文件，编写博客
  2. 输入命令 hexo -g 用来把md文件生存相关的html页面等数据

步骤五 上传本地资源到github上

  1. 在博客目录中，找到配置文件_config.yml

  2. 找到deploy节输入下面的内容

  3. 可选的上传之前输入命令hexo clean清空本地一些缓存数据，以免这些内容也上传

  4. 输入命令npm install hexo-deployer-git –save安装部署器（以后的部署这个就不需要再安装了）

  5. 输入命令hexo d就可以部署了。如果想部署的同时生成html等资源，可以输入命令 hexo d -g
